# Claw Universe Logo V4 - 设计说明

## 设计概述
为 "Claw Universe" 品牌打造的第四代 Logo，在保持 V3 经典布局的基础上进行质感升级。

## 核心改进

### 1. 线条加粗（V4 核心升级）
- **笔画宽度**: 70px（相比 V3 的 40-50px 大幅提升）
- **视觉效果**: 更厚重、更有力量感、更具存在感
- **设计理念**: 粗线条传递稳定、可靠、专业的品牌气质

### 2. 布局结构
```
    ╭──────╮
    │   C   │  ← 蓝色 (#3B82F6)
    ╰──────╯
    
    ╭──────╮
    │   U   │  ← 深灰 (#1F2937)
    ╰──────╯
```
- 保持 C 在上、U 在下的垂直堆叠布局
- 字母间保持 40px 间距，视觉呼吸感充足

### 3. 质感增强
- **圆角端点**: 所有笔画末端采用圆形 cap，消除尖锐感
- **微妙高光**: 顶部添加 subtle 白色渐变叠加，增加立体感
- **底部曲线**: U 的底边带有轻微弧度，更柔和自然

### 4. 配色方案
| 元素 | 色值 | 用途 |
|------|------|------|
| C (上) | `#3B82F6` | 品牌主色 - 科技蓝 |
| U (下) | `#1F2937` | 辅助色 - 沉稳深灰 |
| 背景 | `#FFFFFF` | 纯净白底 |

## 技术规格
- **画布尺寸**: 600 × 800 px
- **笔画宽度**: 70 px
- **文件格式**: PNG (透明背景备选可导出)
- **生成工具**: Python PIL

## 与 V3 对比
| 维度 | V3 | V4 |
|------|-----|-----|
| 笔画宽度 | 40-50px | **70px** |
| 视觉重量 | 中等 | **厚重有力** |
| 质感 | 扁平 | **立体高光** |
| 适用场景 | 网页图标 | **全场景通用** |

## 使用建议
- **主 Logo**: 白底 + 双色字母
- **深色模式**: 可反转为深色背景 + 亮色字母
- **小尺寸**: 70px 笔画确保在 64px 以上仍清晰可辨
- **印刷**: 高对比度配色适合各类印刷介质
