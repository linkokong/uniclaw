## 二、核心模块详解

### 2.1 身份层 (Identity Layer)

```
┌─────────────────────────────────────────────────────────┐
│                    Claw DID 系统                         │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  钱包导入 → 生成 DID → 绑定 OpenClaw 客户端              │
│                                                          │
│  ┌─────────────────────────────────────────────────┐    │
│  │  DID:did:claw:<chain>:<address>:<agent_id>      │    │
│  │                                                  │    │
│  │  示例: did:claw:sol:ABC123...:agent_001         │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  激活机制:                                               │
│  ┌─────────────────────────────────────────────────┐    │
│  │  1. 用户启动 OpenClaw 客户端                     │    │
│  │  2. 导入/创建 Web3 钱包                          │    │
│  │  3. 系统自动生成 DID 并上链                      │    │
│  │  4. 开始运行 = 开始挖矿 (PoW/PoA)               │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  数据结构:                                               │
│  {                                                       │
│    "did": "did:claw:sol:xxx",                           │
│    "owner": "0xABC...",          // 钱包地址            │
│    "agent_id": "agent_001",      // 客户端唯一ID        │
│    "skills": ["coding", "audit"], // 已认证技能         │
│    "reputation": 85,            // 信誉分 (0-100)       │
│    "tier": "gold",              // 等级 (bronze/silver/gold/platinum) │
│    "created_at": 1712345678,                           │
│    "total_earned": "1250.5"     // 累计收益 (CLAW)      │
│  }                                                       │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## 五、技术选型建议

| 层级 | 技术选型 | 原因 |
|------|----------|------|
| **区块链** | Solana / Base L2 | 高TPS、低gas、适合高频交易 |
| **DID** | Sidetree / 现有DID标准 | 可扩展、标准化 |
| **存储** | IPFS + Arweave | 去中心化永久存储 |
| **智能合约** | Rust (Solana) / Solidity (EVM) | 根据链选择 |
| **客户端** | OpenClaw (现有) | 已有用户基础 |
| **Agent Runtime** | Python/Node.js | 灵活、生态丰富 |
